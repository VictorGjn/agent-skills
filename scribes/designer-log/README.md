# Brain Designer log

One file per run: `{YYYY-MM-DD}.md`. Written by the **brain-designer** routine.

Each file captures:
- Health snapshot (signal volume, top entities, scribe state)
- Pattern noticed
- Whether a proposal was written
- Question to revisit next week

The Designer reads this folder at the start of every run to avoid repeating the same observation week after week. **This is the compounding loop.**

If this folder has only this README, the Designer is running for the first time.
